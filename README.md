# Registro GPS Pro

Aplicación PWA para Android que permite:

- Capturar posición GPS.
- Guardar ubicaciones.
- Editar registros.
- Exportar CSV.
- Importar CSV.
- Visualizar ubicaciones en mapa.
- Funcionar offline.

## Instalación local

Abrir:

index.html

## Publicación GitHub Pages

Subir todos los archivos a un repositorio.

Activar:

Settings → Pages

Deploy from Branch

## Publicación Netlify

Arrastrar la carpeta del proyecto a Netlify.

## Instalación Android

Abrir la URL desde Chrome.

Pulsar:

Añadir a pantalla de inicio
